# MyLogger
### 版本: 0.01.000.0001
众嗦粥汁，python有logging， java有log4j2， 可是我一直都只能用printf来输出“日志”，所以，我做了一个非常精简的logger。   
本版本可以使用控制台输出过滤。
## 用法
你甚至可以把它放入include文件夹，你只需要导入进来即可  
如use.c所示：你只需要用init定义输出的名字还有等级即可  
6个函数分别对应6个等级
